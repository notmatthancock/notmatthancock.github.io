  • (0, 0)
  1. Left-click to add a point of "class 0" (indicated by a circle).
  2. Right-click to add a point of "class 1" (indicated by a square).
  3. Click any existing point to remove it.
  4. Start anew by pressing the "Clear" link.
  5. Share or download the data using the respective links.

(Because sometimes you just want a synthetic dataset for testing simple classification problems without having to define the point coordinates functionally.)
